Getting started with HTML

How the World Wide Web Works

To see a web page, our computer sends an HTTP request to a server. That server returns an HTML document to our computer. That document is a web page, which our web browser (Chrome, IE, Firefox, etc.) displays for us.

HTML

HTML stands for Hypertext Markup Language. Most web pages are HTML documents. They can contain a wide range of content formats, from plain text to images, forms, links... Web pages are generally made up of text content and and markup language which dictates the format.

Tags

Tags tell the document how to format or how to treat the text we want displayed. The tags are always surrounded by the symbols "<" and ">", which mean "open instruction" and "close instruction". In most tags, the close ">" is very important, to avoid the formatting affecting the entire document.

Elements

An element is a tag plus its content.

Inline vs Block Elements

Block elements form an "invisible box" around content. Inline elements just change the appearance or action of a part of the content. them.
